#### 问题 8: 粒子系统初始化时序问题
**来源**: [动效光效报告](/workspace/animation-and-lighting-audit-report.md)  
**严重程度**: 中等  
**影响范围**: 首页粒子系统  
**问题描述**: 粒子系统异步加载，但 `initScrollHandler(particles)` 立即调用，可能导致 null 错误
**位置**: `/src/scripts/init.js` 第 30-66 行
**根因分析**: 粒子初始化是异步的，但 scroll handler 在加载完成前就被调用
**建议修复**:
```javascript
function initQiLab() {
  if (initialized) return;
  initialized = true;

  const isHomePage = location.pathname === '/' || location.pathname === '';
  const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches;

  effectsManager.initialize();

  initScrollHandler(null);
  initBackToTop();
  
  if (isHomePage &amp;&amp; !prefersReducedMotion) {
    import('./particles.js').then(({ MinimalParticles }) =&gt; {
      const screenWidth = window.innerWidth;
      const cs = getComputedStyle(document.documentElement);
      const particleCount = screenWidth &lt; 768 ? 40 : screenWidth &lt; 1440 ? 80 : screenWidth &lt; 2560 ? 100 : 120;
      const particleOptions = {};

      if (particles &amp;&amp; particles.canvas) {
        particles.rebuild(particleOptions);
      } else {
        particles = new MinimalParticles('particles-canvas', particleOptions);
        effectsManager.registerEffect('particles', particles, { group: 'background', priority: 10, active: true });
      }
      
      cleanupScrollHandler();
      initScrollHandler(particles);
      cleanupFns.push(() =&gt; particles.destroy());
    });
  }
}
```

### 🟢 低优先级问题 (长期优化)

#### 问题 10: InteractionEnhancements 缺少 cleanup 方法
**来源**: [动效光效报告](/workspace/animation-and-lighting-audit-report.md)  
**严重程度**: 轻微  
**影响范围**: 页面切换时的资源清理  
**问题描述**: `InteractionEnhancements` 类没有提供 `destroy()` 方法来移除事件监听器
**根因分析**: 添加了事件监听器但没有提供清理方法
**建议修复**:
```javascript
class InteractionEnhancements {
  constructor() {
    this.eventHandlers = [];
    this.init();
  }
  
  destroy() {
    this.eventHandlers.forEach(({ element, event, handler, options }) =&gt; {
      element.removeEventListener(event, handler, options);
    });
    this.eventHandlers = [];
  }
}
```
