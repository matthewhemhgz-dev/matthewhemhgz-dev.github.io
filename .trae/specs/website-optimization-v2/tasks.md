# 祈研所网站优化 - 实施计划（分解和优先排序的任务列表）

## [ ] 任务 1：重构 hero-float-card 组件

- **优先级**：P0
- **依赖项**：无
- **描述**：
  - 重构 HeroFloatCard.astro 组件，使其与整体设计风格一致
  - 确保组件在亮色/暗色模式下都能正确显示
  - 优化组件的响应式设计
- **验收标准**：AC-1
- **测试要求**：
  - `human-judgement` TR-1.1：组件显示效果与整体设计风格一致
  - `human-judgement` TR-1.2：组件在亮色/暗色模式下显示正常
  - `human-judgement` TR-1.3：组件在不同设备上响应式显示良好
- **注意**：确保组件的自定义属性接口保持不变，避免破坏现有功能
