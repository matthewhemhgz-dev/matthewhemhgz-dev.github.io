## 任务详情

### T1: 粒子效果恢复与优化

**优先级**: P0 | **负责人**: 前端开发

#### 任务描述
恢复并优化背景光效浮动粒子效果，确保性能和兼容性。

#### 验收标准
- [ ] 粒子效果在首页正常显示
- [ ] 支持中英文版本首页
- [ ] 性能优化（FPS > 55）
- [ ] 移动端自适应（可关闭或降级）
- [ ] 深色/浅色模式适配

#### 相关文件
- [init.js](file:///workspace/src/scripts/init.js)
- [styles/base/tokens.css](file:///workspace/src/styles/base/tokens.css)

#### 技术要求
1. 使用 Canvas API 实现粒子效果
2. 添加性能监控和降级策略
3. 响应式设计，移动端关闭粒子效果
4. 使用设计令牌定义颜色

---

### T2: 页面层次设计规范

**优先级**: P0 | **负责人**: UI/UX设计

#### 任务描述
制定系统化的页面层次设计规范，确保全站视觉一致性。

#### 验收标准
- [ ] 视觉层次规范文档
- [ ] 间距系统完善
- [ ] 字体层级规范
- [ ] 色彩系统优化
- [ ] 组件设计模式

#### 相关文件
- [styles/base/tokens.css](file:///workspace/src/styles/base/tokens.css)
- [styles/base/dark-tokens.css](file:///workspace/src/styles/base/dark-tokens.css)

#### 设计规范要点
1. **间距系统**: 基于8px倍数的间距体系
2. **字体层级**: H1-H6 + 正文 + 辅助文字
3. **色彩系统**: 主色、辅色、中性色体系
4. **阴影层级**: 定义5级阴影深度

---

### T3: 内容分离架构优化

**优先级**: P0 | **负责人**: 架构师

#### 任务描述
完成博客与项目的完整分离，确保信息架构清晰。

#### 验收标准
- [ ] 导航结构清晰区分博客和项目
- [ ] 首页项目展示合理（精选3-4个）
- [ ] 独立项目页面完善
- [ ] 面包屑导航优化
- [ ] Sitemap更新

#### 相关文件
- [Navigation.astro](file:///workspace/src/components/global/Navigation.astro)
- [ProjectsSection.astro](file:///workspace/src/components/sections/ProjectsSection.astro)
- [pages/projects.astro](file:///workspace/src/pages/projects.astro)

#### 架构要点
1. 清晰的路由分离：/blog vs /projects
2. 内容集合设计（使用 Astro Content Collections）
3. 统一的数据模型

### T7: 性能优化与缓存策略

**优先级**: P2 | **负责人**: 性能工程师

#### 任务描述
优化网站性能，提升加载速度和用户体验。

#### 验收标准
- [ ] Lighthouse 评分 > 90
- [ ] 首屏加载时间 < 2s
- [ ] 图片优化（WebP/WebP 格式）
- [ ] 资源懒加载
- [ ] 缓存策略配置

#### 优化方向
1. 图片压缩和格式转换
2. 脚本懒加载
3. 静态资源 CDN 配置
4. Astro 构建优化

---

### T8: 可访问性改进

**优先级**: P2 | **负责人**: 无障碍专家

#### 任务描述
提升网站的无障碍访问性，符合 WCAG 标准。

#### 验收标准
- [ ] WCAG AA 级别通过
- [ ] 键盘导航支持
- [ ] 屏幕阅读器兼容性
- [ ] 对比度符合标准
- [ ] ARIA 属性正确使用

#### 检查要点
1. 颜色对比度检查
2. 焦点状态可见
3. 语义化 HTML
4. 替代文本完善
